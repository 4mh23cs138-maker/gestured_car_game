# Gesture Controlled Car Racing

A real-time gesture-controlled car racing game built with Python, OpenCV, MediaPipe, and Pygame.
Control a racing car using your webcam and hand gestures!

## Requirements
* webcam
* python 3.8+

## Setup
Install the dependencies using pip:
```bash
pip install -r requirements.txt
```

## How to Play
Run the `main.py` script:
```bash
python main.py
```

### Controls
* **Move Hand Left**: Move car left
* **Move Hand Right**: Move car right
* **Open Palm (4 fingers up)**: Accelerate
* **Closed Fist (all fingers down)**: Brake
* **Open Palm (when crashed)**: Restart game

A window will show the webcam tracking your hand. Make sure you are in a decently lit area and your hand is clearly visible!
